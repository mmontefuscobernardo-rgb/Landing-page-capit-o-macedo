# Pasta Pública de Recursos (Assets)

Esta pasta contém os arquivos estáticos acessados diretamente pela raiz do site.

## Como incluir a foto oficial do candidato:

1. Prepare a foto oficial do **Capitão Macedo** em formato JPEG (largura recomendada: 800px a 1200px, proporção ideal quadrada ou vertical 1:1 ou 4:5).
2. Salve o arquivo de imagem exatamente com o seguinte nome:
   `capitao_macedo.jpg`
3. Substitua ou envie o arquivo diretamente para dentro desta pasta (`public/`).
4. Ao fazer o commit e push para o GitHub, o site carregará essa foto automaticamente no cabeçalho e na seção principal de apresentação.
